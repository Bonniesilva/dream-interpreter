# 🌙 Dream Interpreter — OpenClaw 周公解梦技能

> 科学 + 传统双轨解梦，带记忆、日历、月报、语音播报的 AI 解梦助理

## 功能特性

- 🔮 **双轨解读** — 传统周公解梦 + 现代心理学（荣格、弗洛伊德、威胁模拟理论）
- ⚡ **吉凶速判** — 大吉 / 小吉 / 平 / 小凶 / 大凶，有力度不含糊
- 💰 **领域提示** — 财运、感情、事业、健康分领域解读
- 🎙️ **语音播报** — 回复「播报」即可语音朗读解梦结果
- 📅 **梦境日历** — 可视化当月梦境记录，标注吉凶
- 📋 **月度报告** — 统计高频元素、情绪趋势、最值得关注的梦
- 🧠 **规律分析** — 积累3条记录后自动分析你的梦境规律
- 💾 **本地存储** — 所有记录保存在本地，隐私安全

## 安装方法

### 方法一：通过 ClawHub 安装（推荐）

```bash
npx skills install dream-interpreter
```

### 方法二：手动安装

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/dream-interpreter.git

# 复制 Skill 到 OpenClaw
cp -r dream-interpreter ~/.openclaw/skills/dream-interpreter
```

安装完成后重启 OpenClaw Gateway：

```bash
openclaw gateway restart
```

## 使用方法

直接在对话中说：

```
我昨晚梦见一条大蛇追着我跑，最后我飞起来了
```

OpenClaw 会自动识别并触发解梦。

### 触发词

| 场景 | 说法示例 |
|------|----------|
| 解梦 | 「我梦见...」「帮我解梦」「这个梦什么意思」 |
| 语音播报 | 「播报」「读给我听」「语音版」 |
| 查历史 | 「查我的梦境记录」「最近的梦」 |
| 梦境日历 | 「梦境日历」「看看我的梦境日历」 |
| 月度报告 | 「梦境报告」「上个月梦境总结」「生成月报」 |

## 解读框架

### 传统文化层
- 《周公解梦》经典意象
- 民间吉凶说法（含财运意象如粪便、蛇、鱼等）
- 内置 40+ 常见意象数据库

### 科学心理学层
- **荣格原型理论** — 水、火、飞翔、坠落等核心意象解读
- **弗洛伊德** — 潜意识愿望分析
- **威胁模拟理论** — 噩梦的演化意义
- **记忆巩固理论** — REM 睡眠与情绪处理

## 数据存储

梦境记录保存在：
```
~/.openclaw/workspace/memory/dreams/dreams.json
```

数据格式：
```json
{
  "records": [
    {
      "id": "dream_20260328_001",
      "date": "2026-03-28",
      "timestamp": 1743000000,
      "raw": "梦境原始描述",
      "elements": ["蛇", "飞翔"],
      "emotion": "恐惧转平静",
      "verdict": "小吉",
      "domains": ["财运"],
      "summary": "被蛇追逐后飞起，主财运有惊无险"
    }
  ]
}
```
